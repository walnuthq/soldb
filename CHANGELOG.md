# Changelog

All notable changes to this project will be documented here.

## [0.1.0] - 2025

### Added
- Interactive REPL debugger for EVM transactions
- ETHDebug metadata support (Solidity 0.8.29+)
- Source-level stepping, breakpoints, and variable inspection
- Multi-contract debugging
- Transaction simulation mode
- Stylus (Rust smart contract) interop support
- DAP (Debug Adapter Protocol) server
